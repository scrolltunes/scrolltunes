## 10. Developer Handoff Notes

1. **Token System**: All values should reference tokens, not hardcoded colors
2. **Component Architecture**: Use composition for lyric lines (base + state modifiers)
3. **State Management**: Separate Stage state from Practice state to prevent leakage
4. **Performance**:
   - Use CSS transforms for animations (GPU accelerated)
   - Virtualize lyrics list for long songs
   - Pause background animations when tab is hidden
5. **PWA**:
   - Wake lock for performance view
   - Offline support for cached lyrics
   - Full-screen mode support
