## 1. Executive Summary

### 1.1 Current State

The existing `LyricsShareModal` provides:
- Two-step flow (select â†’ customize)
- Gradient backgrounds from album art
- Pattern overlays (dots, grid, waves)
- Basic toggles (shadow, Spotify code, branding)
- Inline text editing
- PNG export via html-to-image

### 1.2 Enhancement Goals

Transform into a full-featured designer with:
- **Multiple layout templates** (10+ curated designs)
- **Drag-and-drop editor** for element positioning
- **Custom fonts** from curated web font library
- **Album art backgrounds** with blur/overlay controls
- **Aspect ratio presets** (1:1, 9:16, 16:9, 4:5)
- **Per-element customization** (colors, sizes, positions)
- **Animated exports** (GIF, MP4/WebM)
- **Desktop-optimized** side-by-side editor layout

### 1.3 Design Principles

1. **Progressive Disclosure** - Simple by default, powerful when needed
2. **Non-Destructive** - All edits reversible, presets don't overwrite custom work
3. **Mobile-First** - Touch-optimized, then enhanced for desktop
4. **Performance** - Smooth 60fps editing, efficient export
5. **Accessibility** - WCAG 2.1 AA compliant throughout

### 7.4 Canvas Scaling Logic

```typescript
function calculateCanvasScale(
  containerWidth: number,
  containerHeight: number,
  canvasWidth: number,
  canvasHeight: number,
  padding: number = 32
): number {
  const availableWidth = containerWidth - padding * 2
  const availableHeight = containerHeight - padding * 2

  const scaleX = availableWidth / canvasWidth
  const scaleY = availableHeight / canvasHeight

  // Fit within container, max 1x
  return Math.min(scaleX, scaleY, 1)
}
```

### 9.3 Animation Rendering Pipeline

```typescript
// For GIF export
async function renderAnimatedGif(
  canvas: HTMLCanvasElement,
  preset: AnimationPreset,
  options: { width: number; height: number; fps: number }
): Promise<Blob> {
  const { width, height, fps } = options
  const frameCount = preset.duration * fps
  const frames: ImageData[] = []

  for (let i = 0; i < frameCount; i++) {
    const progress = i / frameCount
    applyAnimationFrame(canvas, preset, progress)
    frames.push(canvas.getContext('2d')!.getImageData(0, 0, width, height))
  }

  // Use gif.js or similar library
  return encodeGif(frames, { width, height, fps })
}

// For video export (MP4/WebM)
async function renderVideo(
  canvas: HTMLCanvasElement,
  preset: AnimationPreset,
  options: { width: number; height: number; fps: number; format: 'mp4' | 'webm' }
): Promise<Blob> {
  const stream = canvas.captureStream(options.fps)
  const recorder = new MediaRecorder(stream, {
    mimeType: options.format === 'mp4' ? 'video/mp4' : 'video/webm'
  })

  // Record animation frames...
  return new Promise((resolve) => {
    recorder.ondataavailable = (e) => resolve(e.data)
    // Animate and stop after duration
  })
}
```

```typescript
    fps: number

async function exportCard(
  state: DesignerState,
  options: ExportOptions
): Promise<ExportResult> {
  // 1. Prepare canvas at target resolution
  const canvas = await renderToCanvas(state, options.size)

  // 2. Apply pixel ratio for high-DPI
  const pixelRatio = Math.min(window.devicePixelRatio, 3)

  // 3. Generate output based on format
  if (options.format === 'gif' || options.format === 'mp4' || options.format === 'webm') {

  }

  return renderStatic(canvas, options)
```
